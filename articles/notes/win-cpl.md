# Win .cpl

控制面板项的独立可执行程序

### 核心功能与基础设置

| 文件名称 | 功能名称 (中文) | 功能说明 (英文/系统路径) |
| --- | --- | --- |
| **sysdm.cpl** | **系统属性** | System Properties (包含高级系统设置、环境变量、计算机名等) |
| **appwiz.cpl** | **程序和功能** | Programs and Features (卸载或更改程序) |
| **desk.cpl** | **屏幕分辨率 / 显示设置** | Display Settings (在旧版本中为显示属性，现多重定向至新版设置) |
| **main.cpl** | **鼠标属性** | Mouse Properties (设置鼠标主键、指针速度等) |
| **mmsys.cpl** | **声音** | Sound (管理播放、录制设备及系统声音方案) |

---

### 网络与安全

| 文件名称 | 功能名称 (中文) | 功能说明 (英文/系统路径) |
| --- | --- | --- |
| **ncpa.cpl** | **网络连接** | Network Connections (管理网卡、配置 IP 地址和 DNS) |
| **firewall.cpl** | **Windows Defender 防火墙** | Windows Defender Firewall (配置入站/出站规则) |
| **inetcpl.cpl** | **Internet 属性** | Internet Properties (IE 核心的网络代理、安全和隐私设置) |
| **wscui.cpl** | **安全性与维护** | Security and Maintenance (旧称安全中心) |

---

### 硬件、区域与时间

| 文件名称 | 功能名称 (中文) | 功能说明 (英文/系统路径) |
| --- | --- | --- |
| **hdwwiz.cpl** | **设备管理器 / 添加硬件** | Device Manager / Add Hardware (管理和更新驱动程序) |
| **intl.cpl** | **区域** | Region (设置时间、数字格式及非 Unicode 程序的语言) |
| **timedate.cpl** | **日期和时间** | Date and Time (修改时区、同步网络时间) |
| **joy.cpl** | **游戏控制器** | Game Controllers (配置和校准游戏手柄) |
| **powercfg.cpl** | **电源选项** | Power Options (管理睡眠、休眠及电源计划) |

---
